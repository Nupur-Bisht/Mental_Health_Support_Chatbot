# Mental_Health_Support_Chatbot
Built using Python, Streamlit, LangChain, and Retrieval-Augmented Generation (RAG)
This project is an interactive Mental Health Support Chatbot designed to provide users with helpful guidance, emotional support, and general mental health information.
The system uses RAG (Retrieval-Augmented Generation) to give accurate, context-aware responses by retrieving relevant information from a custom knowledge base.
Features
1. RAG-based chatbot
-Uses a vector store + embeddings to fetch the most relevant mental health content before generating responses.

2.Built with Streamlit
-Simple, attractive, and fast web interface for real-time chat.

3.Powered by LangChain
-Utilizes LangChainâ€™s chains, document loaders, and retrieval pipeline.

4.AI-generated conversation
Provides empathetic, supportive replies (not medical advice).
